## 산출 지표

- STT 숫자·단위 정확도 = 정확히 인식된 숫자·단위 수 / 발화한 숫자·단위 총수
- 수정 필요 비율 = 사용자가 수정한 필드 수 / AI가 생성한 전체 필드 수
- 물건 경계 정확도 = 올바르게 분리된 경계 수 / 실제 물건 경계 수
- 평균 처리시간 = 녹음 완료부터 기존 물건목록 등록 완료까지 걸린 시간의 평균
- 등록 성공률 = 최종 등록에 성공한 건수 / 전체 시험 건수

최소 5건, 권장 10건을 수행하기 전에는 “20~30분에서 5~10분으로 단축” 같은 성과 수치를 확정적으로 사용하지 않는다.
